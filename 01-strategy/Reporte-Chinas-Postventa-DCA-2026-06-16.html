§1 · Hallazgos centrales

Lo que la auditoría post-venta de las 3 chinas revela con data DCA contable verificada (no Galindo): el parque acumulado 2025+2026 YTD es de 6,085 unidades nuevas ($2,229M en revenue) repartidas como Changan 2,459 / GAC 1,754 / GWM 1,872. Anexo B · Queries SQL canónicas en D1 bitam-mkt

Reusables para cualquier marca o segmento

-- Q1: Ventas nuevas por mes × marca
SELECT mes, marca, COUNT(*) AS uds, ROUND(SUM(importe_factura),0) AS importe
FROM cubo_30003_ventas
WHERE nuevo_usado = 'AUTOS NUEVOS' AND marca IN ('GAC','CHANGAN','GWM')
GROUP BY mes, marca ORDER BY mes, marca;

-- Q2: Servicio post-venta con valor $ por mes × marca
SELECT mes, marca, COUNT(DISTINCT folio_or) AS or_distintas,
  ROUND(SUM(vta_serv),0) AS serv_$, ROUND(SUM(vta_ref),0) AS ref_$,
  ROUND(SUM(vta_hyp),0) AS hyp_$
FROM cubo_30008_servicio
WHERE marca IN ('GAC','CHANGAN','GWM')
  AND tipo_venta_serv_most IN ('Servicio','Hojalatería')
  AND tipo_orden NOT LIKE '%INTERNA%' AND tipo_orden NOT LIKE 'DEV%'
GROUP BY mes, marca;

-- Q3: Tiempo compra → primer servicio (join VIN limpio)
WITH ps AS (SELECT vin, marca, MIN(fecha_facturacion) AS f
            FROM cubo_30008_servicio
            WHERE tipo_orden NOT LIKE '%INTERNA%' GROUP BY vin, marca),
     v AS (SELECT vin, marca, MIN(fecha_venta) AS f
            FROM cubo_30003_ventas
            WHERE nuevo_usado='AUTOS NUEVOS' GROUP BY vin, marca)
SELECT v.marca, COUNT(*) AS con_servicio,
  ROUND(AVG(JULIANDAY(ps.f)-JULIANDAY(v.f)),0) AS dias_prom
FROM v JOIN ps ON v.vin = ps.vin
WHERE ps.f >= v.f GROUP BY v.marca;
Anexo C · Clasificación de Tipo Orden (cubo 30008)

Heurística de inclusión/exclusión

  • SERVICIO (incluido): Publico, Previa, Express, Reparacion, Garantia, Aseguradora, Hojalateria, Estetica, HYP, Filiales — todo lo que es trabajo al auto del cliente final
  • MOSTRADOR (excluido): VM.*, Mostrador, Mayoreo, Accesorios, Llantas, códigos numéricos — venta de refacciones sin servicio asociado
  • INTERNA (excluido): INTERNA*, INTERNO — movimientos internos del dealer
  • DEV (excluido): DEV*, Devoluciones — operaciones reversas
  • Filtro adicional v3 sugerido: excluir Garantía Planta para análisis de "mantenimiento programado puro" — eliminaría el peak 0-30d sospechoso de PDI
